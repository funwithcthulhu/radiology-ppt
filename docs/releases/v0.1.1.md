# Radiopaedia Case PowerPoint Builder v0.1.1

This release focuses on random-case reliability, review workflow polish, and installer/repository cleanup.

## Highlights

- Random mode searches live Radiopaedia first and avoids cases already used in recent random decks by default.
- Random preparation now refills failed or duplicate picks so a 10-case request should still prepare 10 reviewable cases whenever enough usable public cases are available.
- Random discovery scans a deeper Radiopaedia search window and uses local case history only as a fallback.
- Review workflow includes slower image scrolling, simplified image controls, and more robust keep/remove/replacement behavior.
- Slide rendering preserves image aspect ratio, avoids diagnosis-revealing case-intro text, and handles cases with no selected images without crashing.
- Added a larger edge-case test suite covering parser inputs, contracts, image selection, PowerPoint rendering, random history, and Radiopaedia search parsing.
- Updated the README license badge to avoid stale `invalid` badge output.
