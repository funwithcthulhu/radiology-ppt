# Radiopaedia Case PowerPoint Builder v0.2.2

This patch release focuses on GUI cleanup and navigation consistency.

## Highlights

- Replaced the duplicate top tab strip with a single left-sidebar navigation flow.
- Reordered the main views to `Cases`, `Core Review`, `Library`, and `Activity`.
- Added clear selected-state styling for the left navigation buttons.
- Fixed Library loading so it refreshes whenever the Library view is selected, not only from one specific button path.
- Kept the recent Cases and Core Review layout tightening from the GUI polish pass.
- Updated user-facing docs and release instructions to match the current GUI.

## Validation

- `dotnet build .\csharp\RadiologyPpt.App\RadiologyPpt.App.csproj -c Release`
- `git diff --check`
- UI automation smoke check for sidebar navigation and Library content
- Local Windows installer build and reinstall
