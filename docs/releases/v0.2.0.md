# Radiopaedia Case PowerPoint Builder v0.2.0

This release expands Core Review into its own first-class workflow and cleans up the desktop UI so case conference and CORE review settings are no longer mixed together.

## Highlights

- Added Core Review lecture generation with its own case count, domain, modality mix, output, theme, and question-source controls.
- Added review-stage slide type overrides for Core Review cases, including open diagnosis, multiple-choice diagnosis, pin abnormality, and pin anatomy/structure.
- Updated Core Review counting so NIS and physics questions are included inside the requested total rather than added on top.
- Improved Core Review slide rendering for pin-abnormality prompts, unannotated pins with a movable answer marker, cleaner diagnosis answer choices, and more board-style teaching notes.
- Moved case-conference PowerPoint settings into the Cases tab and removed the old standalone PowerPoint tab.
- Hardened Radiopaedia search/fetch behavior when cached or filtered search pages unexpectedly return empty results.

## Validation
