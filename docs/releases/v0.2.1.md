# Radiopaedia Case PowerPoint Builder v0.2.1

This patch release tightens the desktop GUI after the Core Review workflow changes in v0.2.0.

## Highlights

- Reworked the Cases tab so case-conference settings sit above the case request table and make better use of available space.
- Combined Core Review planning, output, question-source, source-import, and review controls into one integrated panel.
- Improved default window sizing so the app launches fully inside smaller or scaled Windows desktops.
- Made review-window action buttons wrap instead of clipping at narrower widths.
- Fixed small WPF docking/layout issues in the Library and candidate-image review areas.

## Validation

- `npm test`
- `dotnet build .\csharp\RadiologyPpt.App\RadiologyPpt.App.csproj -c Release`
- Local Windows installer build and reinstall
- Live desktop launch smoke check
