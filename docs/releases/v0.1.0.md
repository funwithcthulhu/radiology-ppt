# Radiopaedia Case PowerPoint Builder v0.1.0

Initial Windows installer release.

## Included

- Native Windows desktop app for building Radiopaedia case-based PowerPoints.
- Case request grid for specific diagnoses, random cases, and exact Radiopaedia case URLs.
- Human review window for keeping, skipping, rerolling, re-picking, replacing, or removing images before export.
- Local Library tab for reviewed case history.
- PowerPoint presets, teaching-point slides, and optional Ollama case scoring.
- Core Review PDF import for a private local study workspace.
- Local SQLite state, cache, review decisions, generated output tracking, and backend diagnostics.
- Per-user Windows installer that bundles the app, backend resources, Node runtime, and app dependencies.

## Install

Download and run:

```text
Radiopaedia-Case-PowerPoint-Builder-Setup-v0.1.0.exe
```

The installer defaults to a per-user install and does not require administrator elevation.

## Notes

- The installer is not code-signed yet, so Windows SmartScreen may warn on first install.
- Generated PowerPoints include Radiopaedia attribution.
- Imported Core Review PDFs and generated study data stay local and are not part of the repository.
